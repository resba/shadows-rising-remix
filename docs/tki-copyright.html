  About the NGS project
  Posted by iamsure on July 28, 2002

The Nova Gaming System (NGS) project began as a fork from the BlackNova Traders (BNT) project. The BNT project is a web-based space-themed trading game.

Our goal is to provide a bug-free, non-exploitable game, now deeply evolved from the original BNT code. Moving forward, we also intend to convert the core code to be a framework upon which other game styles can be built.

To accomplish this goal, we will convert the game to a modular, event-driven system, with independent language and sql files. In addition, each game will itself also be theme-able/skinnable.

Both BNT and NGS are copyrighted by each of the developers listed below, for each of their respective contributions, and placed under the trust of the lead developers (L. Patrick Smallwood and Ron Harwood respectively for NGS / BNT).
