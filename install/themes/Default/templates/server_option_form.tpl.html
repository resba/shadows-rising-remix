{$app_name} Install Utility

This section will install all the database tables required to allow the Shadows Rising RPG Game Engine to operate. As of version 0.0.4 this section will also install default tables to allow access to the default Shadows Rising Game which is in development. Over time, Game Installation will be transferred to a separate section.

Please complete the options below. For those upgrading from prior versions, you may wish to keep your current list of registered players and users. To facilitate this you may select the "Retain User Accounts?" option below. The Administrator Password you enter is the password that will be set for the default "Admin" account.

It is strongly suggested you backup your current database before proceeding!
